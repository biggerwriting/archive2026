# Mind Map Drawing

一个纯前端的脑图展示程序，支持把 Markdown 层级文本渲染成可交互思维导图。

## 支持能力

- 解析 `# / ## / ### / -` 的层级结构
- 自动左右分布脑图分支
- 画布拖拽、缩放、自适应
- 点击节点展开/收起
- 全部展开 / 收起到二级
- 关键词搜索高亮

## 快速使用

直接用浏览器打开 `index.html`。

## 输入格式

```md
# 根节点
## 一级分类
### 二级分类
- 叶子节点
- 叶子节点
```

把你的完整课程列表粘贴到左侧输入框，点击“渲染脑图”即可。

## 技术实现

- `index.html`: 页面结构
- `styles.css`: 样式与布局
- `app.js`: 文本解析、D3 渲染、交互控制
