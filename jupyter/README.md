# 启动 Jupyter

在这个项目目录下运行：
```bash
cd /Users/tongqianwen/ExpProjects/learn/jupyter
source venv3/bin/activate
jupyter lab
```

▎ 也可以用经典界面：jupyter notebook（更简单），或 jupyter lab（功能更强）

启动后浏览器会自动打开，显示文件列表，点击任意 .ipynb 文件即可打开。

---
基本操作

两种模式

┌──────────┬────────────────┬────────────────────────────────┐
│   模式   │      标志      │              作用              │
├──────────┼────────────────┼────────────────────────────────┤
│ 命令模式 │ 单元格边框蓝色 │ 管理单元格（新增、删除、移动） │
├──────────┼────────────────┼────────────────────────────────┤
│ 编辑模式 │ 单元格边框绿色 │ 编写代码                       │
└──────────┴────────────────┴────────────────────────────────┘

按 Esc → 命令模式，按 Enter → 编辑模式

最常用快捷键

┌───────────────┬───────────────────────────────┐
│    快捷键     │             功能              │
├───────────────┼───────────────────────────────┤
│ Shift + Enter │ ▶️ 运行当前单元格，跳到下一个 │
├───────────────┼───────────────────────────────┤
│ Ctrl + Enter  │ ▶️ 运行当前单元格，留在原地   │
├───────────────┼───────────────────────────────┤
│ A             │ 在上方插入新单元格            │
├───────────────┼───────────────────────────────┤
│ B             │ 在下方插入新单元格            │
├───────────────┼───────────────────────────────┤
│ DD            │ 删除当前单元格                │
├───────────────┼───────────────────────────────┤
│ M             │ 切换为 Markdown 文本          │
├───────────────┼───────────────────────────────┤
│ Y             │ 切换为 Code 代码              │
├───────────────┼───────────────────────────────┤
│ Ctrl + S      │ 保存                          │
└───────────────┴───────────────────────────────┘
