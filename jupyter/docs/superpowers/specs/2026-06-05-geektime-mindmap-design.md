## 1. 需求概述

构建一个**单 HTML 文件**的交互式脑图工具，用于管理极客时间 185+ 门课程。核心功能：

- 以可折叠横向树形脑图可视化所有课程分类
- 对课程打状态标签、自定义标签、备注
- 搜索 + 多维过滤
- 导出 / 导入 JSON，持久化学习进度

---

## 2. 技术选型

| 项目 | 选择 | 理由 |
|------|------|------|
| 实现形式 | 单文件 HTML | 零依赖、双击即用、可离线 |
| 可视化库 | D3.js v7（CDN） | 对节点渲染完全掌控，动画流畅，横向树内置支持 |
| 样式 | 内嵌 CSS，暗色主题 | 无外部样式表依赖 |
| 数据 | 内嵌 JS 常量 + JSON 导入覆盖 | 首次打开即可用，无需服务器 |

## 4. 数据模型（JSON Schema）

```json
{
  "meta": {
    "title": "极客时间课程脑图",
    "version": "1.0",
    "exportedAt": "2026-06-05T10:00:00Z"
  },
  "customTags": [
    {
      "id": "tag_01",
      "label": "下周必看",
      "color": "#a78bfa",
      "emoji": "🔥"
    }
  ],
  "categories": [
    {
      "id": "cat_lang",
      "name": "编程语言",
      "icon": "💻",
      "courses": [
        {
          "id": "lang_01",
          "name": "Go 语言核心36讲",
          "group": "Go",
          "status": "important",
          "customTagIds": ["tag_01"],
          "note": "第3周开始学，重点看并发章节",
          "startedAt": "2026-06-10",
          "doneAt": null
        }
      ]
    }
  ],
  "stats": {
    "total": 185,
    "important": 3,
    "learning": 5,
    "done": 12,
    "deleted": 8
  }
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `status` | `"normal" \| "important" \| "learning" \| "done" \| "deleted"` | 内置状态，互斥 |
| `customTagIds` | `string[]` | 可叠加多个自定义标签（与 status 并行） |
| `note` | `string` | 自由备注文本 |
| `startedAt` | `string \| null` | 状态变为 learning 时自动记录 |
| `doneAt` | `string \| null` | 状态变为 done 时自动记录 |

## 5. 节点视觉规则

### 颜色对照

| 状态 | 圆点颜色 | 文字颜色 | 说明 |
|------|----------|----------|------|
| `normal` | `#475569` 灰 | `#94a3b8` | 未标注 |
| `important` | `#f59e0b` 金 | `#fbbf24` | ⭐ 重点 |
| `learning` | `#3b82f6` 蓝 | `#60a5fa` | 📚 学习中 |
| `done` | `#22c55e` 绿 | `#4ade80` | ✅ 已完成 |
| `deleted` | `#ef4444` 红 | 删除线 | 🗑️ 默认隐藏 |

### 节点类型

- **根节点**：金色大圆，居左，始终显示
- **分类节点**：彩色圆角矩形（每类一色），显示 icon + 名称 + 课程数，点击展开/收起
- **课程节点**：小圆点 + 文字 + 状态图标，有自定义标签时节点带标签色描边

## 6. 交互行为

### 画布操作

| 操作 | 行为 |
|------|------|
| 单击分类节点 | 展开 / 收起该分类 |
| 左键单击课程节点 | 弹出操作菜单 |
| 右键单击课程节点 | 弹出操作菜单 |
| 鼠标悬停课程节点 | Tooltip：课程名 + 备注摘要 + 自定义标签 |
| 滚轮 | 缩放画布 |
| 拖拽空白处 | 平移画布 |
| 双击空白处 | 重置到初始视图 |

### 右键操作菜单

```
├─ ⭐ 标记重点 / 📚 学习中 / ✅ 已完成  （点击切换，再点取消）
├─ ─────────────
├─ 🏷️ 管理自定义标签  →  已有标签勾选 + [+ 新建标签]
├─ 📂 移动到分类      →  18 个分类子菜单
├─ 📝 添加备注        →  小型浮层文本框
├─ ─────────────
├─ 🗑️ 删除课程
└─ ↩️ 恢复默认
```

### 自定义标签创建

1. 右键菜单 → 「管理自定义标签」→「+ 新建标签」
2. 弹出浮层：输入标签名 + 选择颜色（6 个预设色）+ 选择 emoji（可选）
3. 创建后立即应用到当前课程，同步出现在左侧过滤面板

## 7. 搜索与过滤

### 搜索框

- 实时匹配（输入即生效）
- 匹配范围：课程名 + 备注内容
- 匹配结果：高亮节点 + 自动展开所在分类 + 非匹配节点变灰
- 清空：点 ✕ 或 Esc 恢复

### 过滤逻辑

| 维度 | 运算 |
|------|------|
| 状态过滤 ∩ 分类过滤 | AND |
| 多个自定义标签 | OR（带任意一个即显示） |
| 搜索 ∩ 过滤 | AND |
| 「显示已删除」开关 | 独立控制，默认关闭 |

## 8. 导入 / 导出

### 导出

- 点击「📤 导出 JSON」
- 生成完整 JSON，文件名：`geektime_mindmap_YYYY-MM-DD.json`
- 包含：所有状态、自定义标签定义、备注、时间戳、统计快照

### 导入

1. 点击「📥 导入 JSON」→ 文件选择器（`.json`）
2. 解析成功后弹出确认对话框，显示上次导出时间和标注统计
3. 用户选择导入方式：

| 模式 | 行为 |
|------|------|
| **合并**（默认） | 以导入文件为准；当前有、导入没有的标注保留 |
| **覆盖** | 完全替换为导入文件状态 |

> **分类归属说明**：课程通过「移动到分类」操作后，其在 JSON 中物理移入新分类的 `courses` 数组，原分类不再包含该课程。导入时以文件中的分类归属为准（两种模式相同）。

### 容错

- JSON 格式非法 → 提示错误，不导入
- 版本号不匹配 → 警告后可继续
- 导入文件含未知课程 ID → 忽略
